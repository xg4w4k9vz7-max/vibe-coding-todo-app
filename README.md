# Vibe Coding Todo App

This project is a functional Todo List application developed as part of a homework assignment exploring the concept of **vibe coding** and AI-assisted development tools.

The goal of the project is to understand how AI-first coding tools can support software development beyond traditional code completion.

## Features
- Add new tasks
- Delete existing tasks
- Persistent storage using LocalStorage
- Simple and clean user interface
- Fully functional without any external dependencies

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)
- Cursor (AI-first code editor)

## Vibe Coding Tool Used
**Cursor** was used as the primary vibe coding tool during development. It provided context-aware code suggestions, refactoring assistance, and natural language interaction inside the IDE.

## Installation and Setup
1. Clone this repository
2. Open the `index.html` file in any modern web browser
3. Start adding and managing your tasks

No additional setup or dependencies are required.

## How to Run
Simply double-click `index.html` or open it via a browser.
